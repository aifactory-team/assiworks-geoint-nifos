# 🌲 AssiWorks - NIFOS — 산림 위성 변화 모니터링

AssiWorks - NIFOS 에이전트가 한국의 **산림 권역**(설악·강원영동·울진·지리산·제주 한라산·북한복구대상지 등)을 win_km 타일로 자동 순환하며, 가장 최신 Sentinel-2(ESA, 10m) 위성영상에서 **산불 피해·병해충 고사·산사태·산지훼손·산림건강성** 변화를 분석하고, 같은 타일의 과거 같은 계절과 비교 검증해 **의미 있는 변화 신호일 때만** 발행하는 리포트입니다. 국립산림과학원 농림위성 활용 산출물 체계(산림재해·산림생태/건강성·산림자원/통계)를 공개위성으로 재현했습니다. 하루 여러 건을 **날짜/시각 슬롯**으로 발행합니다.

> ⚠️ **추정치·공개위성 안내**: 본 콘텐츠는 공개된 Sentinel-2(ESA Copernicus) 위성영상을 AI·알고리즘이 자동 분석한 **추정 결과**로, 사실과 다를 수 있습니다. Sentinel-2(10m)는 국립산림과학원 농림위성(5m, Red-Edge·NIR)의 공개 프록시로 사용한 참고용이며, 산불은 실시간 화점이 아니라 **산불 이후 피해지·식생손실** 탐지입니다. 모든 판정은 임상도·국가산림자원조사·현장조사·정밀 판독을 대체하지 않습니다.

발견(타일에 걸치는 최신 1장) → 분야별 지표 분석(산불 NBR·병해충 NDRE·산사태 BSI·산지훼손 NDVI손실·건강성 NDVI편차) → 과거 같은 계절 시계열 이상(z) 검증 → **평년·무변화는 발행하지 않음**. 모든 판정은 '그 타일 자체 과거 대비 변화'로, 상시 산림 배경은 자동 배제하고 *신규 변화*만 포착합니다.
